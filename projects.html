Smart Shoppers

Developed a shopping app using the Model-View-Controller (MVC) design pattern to enhance project organization and modularity. Designed and implemented six controller classes (CartController, CategoryController, DBController, UserController, StoreController) to manage different aspects of the application. Constructed the model layer with classes for Admin, Category, Item, Store, and User to define the data structure. Created a comprehensive view layer with tables and user interface components for a seamless user experience. Applied the MVC pattern based on the observer pattern to enable automatic updates in the view upon data modifications. Developed multiple helper methods to reduce code repetition and enhance code reusability. Implemented a Database Controller to manage CSV and storage operations efficiently. Integrated the shopping cart functionality with all relevant components, allowing users to add, modify, and delete items in their cart. Corrected UML and project documentation to reflect the relational dependencies between components accurately. Introduced a searchable interface to improve the search functionality within the application. Ensured a clear separation of user interface and logic, adhering to best practices in software design.
